Documents Requis

Dossiers nécessaires

Sélectionnez votre situation professionnelle pour voir les documents requis.

👔 Documents — Salarié

  • ✓ Attestation de travail
  • ✓ Trois (03) fiches de paie
  • ✓ Affiliation CNAS
  • ✓ Relevé de compte CCP ou bancaire (3 derniers mois)
  • ✓ Attestation de retrait de compte euro
  • ✓ Attestation de paiement en espèce (si salaire reçu en espèce)
  • ✓ Ordre de mission (en cas de visa d'affaires)

🏪 Documents — Commerçant

  • ✓ Registre de commerce
  • ✓ C20 + Extrait de rôle
  • ✓ CASNOS
  • ✓ Relevé de compte personnel ou commercial (3 derniers mois)
  • ✓ Attestation de retrait de compte euro

🛠️ Documents — Artisan

  • ✓ Carte artisan
  • ✓ C20 + Extrait de rôle
  • ✓ CASNOS
  • ✓ Relevé de compte dinar (3 derniers mois)
  • ✓ Attestation de retrait de compte euro

💡 Documents — Auto-entrepreneur

  • ✓ Carte d'auto-entrepreneur
  • ✓ C20 + Extrait de rôle
  • ✓ CASNOS
  • ✓ Relevé de compte dinar (3 derniers mois)
  • ✓ Attestation de retrait de compte euro

🌾 Documents — Agriculteur

  • ✓ Carte d'agriculture
  • ✓ Attestation d'agriculteur
  • ✓ Fiche signalétique
  • ✓ CASNOS
  • ✓ Preuve des revenus annuels (Bilan)
  • ✓ Relevé de compte dinar (3 derniers mois)
  • ✓ Attestation de retrait de compte euro

⚠️ La liste peut varier selon le pays de destination et le type de visa. Contactez-nous pour une liste personnalisée.

Questions fréquentes

Quels pays traitez-vous ?

France, Allemagne, Espagne, Italie, USA, Canada, Royaume-Uni, Qatar, Azerbaïdjan, Malte, Grèce et bien d'autres. Contactez-nous pour votre destination.

Quel est le délai de traitement ?

Les délais de traitement varient en fonction du pays concerné et du type de visa demandé.

Garantissez-vous l'obtention du visa ?

Nous ne pouvons pas garantir l'obtention (la décision revient aux consulats), mais nous maximisons vos chances avec un dossier complet et bien préparé.

Comment se passe le paiement ?

Nos tarifs sont transparents. Le mode de paiement est discuté lors de votre premier contact. Pas de frais cachés.

Quels sont vos horaires ?

Disponibles 7j/7 sur WhatsApp. Pour les rendez-vous physiques, contactez-nous pour convenir d'un créneau.
